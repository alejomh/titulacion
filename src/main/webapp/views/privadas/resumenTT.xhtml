Tener en cuenta las siguientes normas:

  • - El resumen debe contener desde 500 hasta 1000 palabras, a excepción de los que realizan atículo científico (el resumen puede tener la cantidad de palabras según las normativas de la revista).
  • - Separe las palabras claves con comas ( , )
  • - Debe utilizar el acento gráfico agudo ( ´ ), más no el acento gráfico grave ( ` )
  • - Debe usar un solo espacio entre palabras
